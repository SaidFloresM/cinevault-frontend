 CineVault 

Catálogo de películas fullstack desarrollado como parte del periodo de prueba en Turing Inteligencia Artificial.

# Tecnologías

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Supabase (Base de datos + Edge Functions)

# Funcionalidades

- Registro e inicio de sesión con JWT
- Catálogo de 16+ películas en 8 géneros
- Filtro por género en tiempo real
- Paginación del lado del servidor
- Panel de administración con CRUD completo
- Roles de usuario: admin y user
- Diseño responsivo (mobile + desktop)

# Instalación

1. Clonar el repositorio
2. Instalar dependencias: npm install
3. Configurar variables de entorno en .env
4. Ejecutar: npm run dev
